📊 Customer Churn Analysis & Retention Strategy
🚀 Project Overview

This project focuses on analyzing customer churn in a telecom dataset and identifying key factors influencing customer retention.

The goal was not just to analyze churn, but also to propose actionable strategies to retain customers based on data-driven insights.

📁 Dataset Summary
Total Customers: 7,043
Churned Customers: 1,869
Retained Customers: 5,174
Total Features: 21
Key Features:
Numerical: Tenure, Monthly Charges, Total Charges
Categorical: Contract Type, Payment Method, Internet Service, Tech Support, etc.


🛠️ Tools & Technologies Used
Python → Data Cleaning & EDA
SQL (PostgreSQL) → Data Analysis & Querying
Power BI → Interactive Dashboard & Visualization

🔍 Data Cleaning & Preparation
Handled missing values using appropriate methods (mean/mode)
Verified no duplicate records
Standardized column names for better readability
Converted data types for proper analysis

📊 Exploratory Data Analysis (EDA)
Performed detailed analysis to understand:
    -Customer demographics
    -Service usage patterns
    -Revenue distribution
    -Churn behavior across different segments


📈 Dashboard Insights (Power BI)
🔑 Key Findings:
📉 High Churn in Month-to-Month Contracts
Customers with month-to-month contracts have the highest churn rate (~42%)
💰 Higher Charges = Higher Churn
Churned customers pay significantly higher monthly charges
⏳ Low Tenure Customers are High Risk
New customers are more likely to churn
🛡️ Lack of Support Services Increases Churn
Customers without:
Tech Support
Online Security
are more likely to leave
💳 Payment Method Matters
Customers using electronic check show higher churn compared to automatic payment methods


🧠 Retention Strategy (My Approach)
In this project, I went beyond analysis and tried to solve the business problem by suggesting retention strategies:
✅ 1. Encourage Long-Term Contracts
Offer discounts on yearly and 2-year plans
Reduce dependency on month-to-month customers

✅ 2. Optimize Pricing Strategy
Provide better value for high-paying customers
Bundle services to justify cost

✅ 3. Focus on New Customers
Improve onboarding experience
Provide early engagement offers

✅ 4. Promote Support Services
Increase awareness of:
Tech Support
Online Security
Device Protection

✅ 5. Improve Payment Experience
Encourage automatic payments
Reduce friction in billing process



📌 Key Business Impact
By implementing these strategies, companies can:
                                 -Reduce churn rate
                                 -Increase customer lifetime value
                                 -Improve customer satisfaction
